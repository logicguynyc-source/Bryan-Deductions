How to Use

1. Set up — Enter comma-separated premises and a conclusion, then click Create Proof.

2. Type formulas — Each row has a formula cell (left) and a justification cell (right). Click either to edit.

3. Justifications — Type the rule abbreviation followed by line numbers, e.g. ∧I 1,2 or →E 3,4. Premises are labelled automatically.

4. Subproofs — Use the +sub button to open a subproof (needed for →I, ¬I, ∨E). The first line of a subproof is always an assumption.

5. Check — Click Check Proof to validate. Each line turns green (valid) or red (error).

Symbols — Use the symbol bar above, or type: - for ¬, /\ for ∧, \/ for ∨, -> for →, <-> for ↔, _|_ for ⊥.
