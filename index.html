Inference Rules
How to Use

1. Set up — Enter comma-separated premises and a conclusion, then click Create Proof.

2. Type formulas — Each row has a formula cell (left) and a justification cell (right). Click either to edit.

3. Justifications — Type the rule abbreviation and line numbers, e.g. ∧I 1,2 or →E 3,4. Premises are labelled automatically.

4. Subproofs — Use +sub to open a subproof (needed for →I, ¬I, ∨E, ∀I, ∃E). The first line is always an assumption.

5. FOL quantifiers — Write ∀x Fx or ∃x Gxy. For ∀E and ∃I specify the substitution: ∀E 1 [x/a] (replace x with a). For ∀I write ∀I 3-5 [c/x] (c is the arbitrary constant, x the variable). For ∃E write ∃E 1 2-4.

6. Check — Click Check Proof to validate. Each line turns green (valid) or red (error).

TFL shortcuts — - or ~ for ¬, /\ for ∧, \/ for ∨, -> for →, <-> for ↔, _|_ for ⊥.

FOL shortcuts — See the FOL Syntax panel for formula syntax and substitution notation.

FOL Syntax Guide
Atomic sentences
PTFL sentence letter
FaPredicate F, term a
GabRelation G, terms a, b
a=bIdentity
Predicates: single uppercase letter (not A or E alone).
Terms / constants: single lowercase letter a–w.
Variables: single lowercase letter a–z (used inside quantifiers).
Writing quantifiers
You typeMeans
∀x FxFor all x: Fx
forall x FxSame (word form)
all x FxSame (word form)
∃x FxThere exists x: Fx
exists x FxSame (word form)
some x FxSame (word form)
Use parentheses to delimit scope when needed: ∀x (Fx → Gx).
Without parens, the quantifier takes the rest of the line as its scope.
Note: Ax/Ex shorthands are not supported — they are ambiguous with predicate applications like Ea (predicate E applied to a).
Justification notation
RuleWriteExample
∀E ∀E m [x/t] From ∀x Fx on line 1,
derive Fa: write ∀E 1 [x/a]
∃I ∃I m [t/x] From Fa on line 2,
derive ∃x Fx: write ∃I 2 [a/x]
∀I ∀I m [c/x] From Fc on line 3 (c arbitrary),
derive ∀x Fx: write ∀I 3 [c/x]
c must not appear in any premise or undischarged assumption. [c/x] can often be omitted.
∃E ∃E m i–j From ∃x Fx on line 1 and
subproof on lines 2–4: write ∃E 1 2-4
[x/t] means "substitute variable x with term t". The substitution spec can often be omitted — the checker will infer it automatically for ∀E and ∃I.
